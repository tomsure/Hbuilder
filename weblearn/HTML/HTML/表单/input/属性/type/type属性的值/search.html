但type=search会有许多默认样式和行为，这次开发遇到的有： 例如： 1、会默认下拉框显示搜索历史记录; 2、输入时自动弹出“x”，“x”的样式在不同手机上，样式不同； 3、IOS 手机（测试时为iphone6 ios10）上输入框为椭圆形. 但我们希望样式按照我们自定义的样式显示，并且各个手机上能统一： 1、设置input autocomplete="off"去掉弹出的下拉框； 2、将默认的“x”隐藏掉： 在CSS中设置input[type="search"]::-webkit-search-cancel-button{ display: none; } 3、针对ios 设置样式, 去除ios下input 椭圆形: -webkit-appearance: none; 4、如果提交搜索时想使用ajax，那么可以阻止表单的默认行为： container.on('submit', '.input-kw-form', function(event){ event.preventDefault(); })
