灾害发生后，党中央、国务院高度重视。中共中央总书记、国家主席、中央军委主席习近平立即作出重要指示，要求福建省和相关部门迅速组织力量开展抢险救援，全力搜救被困、失踪人员，尽最大努力减少人员伤亡，并妥善做好伤员救治、伤亡人员亲属安抚等善后工作。加强监测预报，注意科学施救，防止发生次生灾害。国务院有关部门要指导地方和企业加强各类灾害和安全生产隐患排查，强化责任落实，确保人民群众生命财产安全。
