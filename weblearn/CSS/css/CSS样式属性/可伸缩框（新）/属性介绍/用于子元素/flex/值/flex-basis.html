flex属性的第二个值用于设置flex-basis，用于设置定义弹性盒子元素的默认基准值，当子元素没有设置宽度，且第一个值都为1的时候，则子元素的宽度就会等比划分父元素的宽度， 然后当设置这个值之后，就相当于给每个元素设置了一个计算最后宽度（或高度）的初始宽度，例如这里父元素dv宽度为950px，每个子元素设置了flex-grow的值都为1，设置flex-basis的值分别为200px,300px,250px,这时候，这三个值加起来是750px，而父元素的值是950px，则多余的200px宽度就会根据flex-grow的值进行按比例分配，如果flex-grow都为1的话，则就是200px/3,然后分配给三个子元素，最后得出子元素最后的值就为266.67px、366.67px、316.67px， 如果如果三个flex-grow不为1的话就按比例分配，这个也就是按比例分配多出来的200px宽度，分配给每个子元素的初始值，得到每个子元素最后的宽度；（例如：如果dv2 的flex属性的第一个值设为2，其他两个子元素的flex第一个值都为1，则就是把这200px分成四份，dv2的初始宽度就会加上其中的两份，得到最后的宽度，其他两个元素各得到其中的一份再加上初始宽度，得到最后的宽度） 收缩时候的计算方法： 收缩也就是子元素设置的初始值之和大于父元素的时候，就需要收缩； dv1的宽度： ((200px*1)/750)200=53.33; //计算收缩值 200-53.33=146.67;//使用初始值减去收缩值，达到最后的宽度 dv2的宽度： ((200px*1)/750)*300=80px //计算收缩值； 300-80=220px； dv3的宽度： ((200px*1)/750)*250=66.67// 250-66.7=183.33
