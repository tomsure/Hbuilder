以下样式可用于a, button, 或 input 元素上：

一个按钮可以添加多个类达到综合效果

类名 描述 实例
.btn 为按钮添加基本样式
.btn-default 默认/标准按钮
.btn-primary 原始按钮样式（未被操作）
.btn-success 表示成功的动作
.btn-info 该样式可用于要弹出信息的按钮
.btn-danger 表示一个危险动作的按钮操作
.btn-warning 表示需要谨慎操作的按钮
.btn-link 让按钮看起来像个链接 (仍然保留按钮行为)
.btn-lg 制作一个大按钮
.btn-sm 制作一个小按钮
.btn-xs 制作一个超小按钮
.btn-block 块级按钮(拉伸至父元素100%的宽度)
.disabled 禁用按钮
